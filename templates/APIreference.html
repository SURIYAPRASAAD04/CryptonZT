Core Endpoints

POST
/api/protectKey
Encrypts text or files using quantum-resistant algorithms. Returns the encrypted data ID and metadata.

Request Parameters

Parameter Type Description
datarequired String/File The text content or file to be encrypted
metadata String Optional description for the encrypted data

Example Request

JSON Payload
{
  "data": "Sensitive information to protect",
  "metadata": "API Test"
}

Example Response

Success 201
{
  "data_format": "txt",
  "data_id": "DATA_f0d1d5c7cc42ead9",
  "key_id": "KEY_d8ae58a4ed9934fd",
  "metadata": "API Test",
  "size_mb": 0.0,
  "status": "success",
  "timestamp": "2025-08-10T00:32:47.943033"
}

DELETE
/api/data/:data_id
Permanently deletes encrypted data and all its fragments. This action is irreversible.

Example Response

Success 200
{
  "status": "success",
  "message": "Data and all fragments permanently deleted"
}

Key Management

GET
/api/keys/:key_id
Retrieves metadata about an encryption key, including its algorithm and associated data.

Example Response

Success 200
{
  "key": {
    "algorithm": "Kyber1024-AES256",
    "associated_data": {
      "data_format": "txt",
      "meta_data": "Management Test"
    },
    "created_at": "2025-08-10T01:08:00.572000",
    "data_id": "DATA_01c3c7179e42f47b",
    "key_id": "KEY_1368e0b0bf325f38",
    "status": "active"
  },
  "status": "success"
}
POST
/api/keys/rotate/:data_id
Rotates the encryption keys for specified data while maintaining accessibility.

Example Response

Success 200
{
  "status": "success",
  "message": "Key rotation completed",
  "new_key_id": "KEY_9k3l5m7n8p",
  "data_id": "DATA_01c3c7179e42f47b"
}

Data Management

GET
/api/manageData
Lists all encrypted data items with their metadata. Supports pagination.

Query Parameters

Parameter Type Description
page Integer Page number for pagination. Default: 1
limit Integer Items per page. Default: 20, Max: 100
sort String Sort field (encrypted_at, data_format). Default: encrypted_at

Example Response
